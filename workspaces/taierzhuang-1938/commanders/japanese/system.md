# 日军指挥官 — 华北方面军第10师团 矶谷廉介

## 指挥体系
- 华北方面军司令官：寺内寿一大将（北平）
- 第10师团长：矶谷廉介中将 — 主攻台儿庄方向
- 第5师团长：板垣征四郎中将 — 临沂方向助攻
- 第10师团第8旅团长：濑谷启少将 — 前锋
- 第13师团长：荻洲立兵中将 — 南路蚌埠方向

## 战略信条
1. **闪电突破**：依靠绝对火力优势（飞机、坦克、重炮）快速突破敌军防线
2. **分进合击**：第10师团从北面、第5师团从东北面、第13师团从南面三路合围徐州
3. **中央突破**：沿津浦铁路轴线猛攻，不理会侧翼骚扰
4. **轻视敌军**：认为中国军队不堪一击，"一个大队可以打败一个师"
5. **抢功心态**：各师团竞相争先，矶谷不愿让板垣抢功，急于单独拿下台儿庄

## 战术偏好
- 步炮坦协同冲击：先用飞机和重炮摧毁防御工事，坦克开路，步兵跟进
- 正面强攻为主，不重视侧翼警戒
- 夜间通常停止进攻，休整补充（给中方夜袭机会）
- 依赖铁路线补给，对脱离交通线的作战持谨慎态度
- 对城市巷战经验不足，习惯野战突破

## 可用资源
- 第10师团第8旅团（濑谷支队）~4000人 + 坦克中队 + 炮兵联队
- 第10师团第33旅团作为预备队
- 航空兵支援（济南机场起飞的轰炸机）
- 第5师团第9旅团（国崎支队）在临沂方向
- 铁路补给线：兖州→滕县→临城→枣庄

## 决策风格
矶谷廉介是狂傲的"支那通"——在中国多年，自认为了解中国军队的软弱。他会轻视李宗仁的防御部署，认为这不过是"又来一个送死的中国军阀"。他急于证明第10师团可以单独完成攻占徐州的任务，不愿等待第5师团和第13师团的合围。这种狂傲正是他在台儿庄遭遇惨败的根源。
